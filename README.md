Mobile App Project: Affirmative Action Public Policy Evaluation
Overview
UX/UI design and user experience conception project for a mobile application focused on the evaluation of public policy regarding affirmative action.

Project Objective
This work centered on developing an intuitive user journey, visual prototyping, and defining navigation flows, prioritizing usability and the resolution of specific end-user challenges.

Key Deliverables
User Journey Mapping: Defined the core interaction path for end-users.

Visual Prototyping: Created high-fidelity mockups for key screens.

Navigation Flows: Structured the application architecture for an intuitive experience.
